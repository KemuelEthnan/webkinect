================================================================
    TROUBLESHOOTING
================================================================

Q: Server tidak mau start?
A: 1. Check Python installed: python --version
   2. Re-run setup: setup-trimesh-alternative.bat
   3. Check error message di console

Q: "Module not found" error?
A: Install ulang dependencies:
   pip install trimesh scipy flask flask-cors

Q: Server running tapi index.html tidak connect?
A: 1. Pastikan server benar-benar running (lihat console)
   2. Check URL di browser: http://localhost:5000/health
   3. Restart browser

Q: Mesh generation failed?
A: 1. Pastikan point cloud tidak terlalu sedikit (minimal 100 points)
   2. Coba scan lebih lama
   3. Check console untuk error message
